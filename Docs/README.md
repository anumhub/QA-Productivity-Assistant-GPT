# QA Productivity Assistant (Custom GPT)

A Custom GPT designed to help manual and hybrid QA engineers with requirement analysis, test design, test data suggestions, and basic defect triage.

## Problem
Manual test planning and test case design are time-consuming and inconsistent across teams.

## Solution
This Custom GPT embeds a QA analyst mindset and structured output formats so testers can use short prompts without repeating context.

## Key Capabilities
- Requirement analysis and risk identification
- Structured functional test case generation
- Edge case and negative scenario coverage
- Test data suggestions
- Bug report improvement + severity/priority guidance
- Test coverage gap detection

## Project Artifacts
- `docs/` – Setup guide (how to create and configure the Custom GPT)
- `prompts/` – 10 sample prompts to demonstrate real usage
- `screenshots/` – GPT configuration screenshot(s) and sample outputs

## How to Use
Use the sample prompts in `prompts/sample-prompts.md` and provide your feature requirement, user story, or defect report.
