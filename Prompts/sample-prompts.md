# Sample Prompts

1. Requirement analysis: Analyze the following requirement and identify testable areas and risks.
2. Test cases: Generate functional test cases in tabular format for this requirement.
3. Edge cases: What edge cases and boundary conditions should be tested?
4. Test Data: Suggest test data combinations for the above test cases.
5. Regression scope : Which areas should be included in regression testing and why?
6. Exploratory Testing: Suggest exploratory testing ideas for this feature.
7. Bug Report Improvement: Improve the clarity and completeness of the following bug report.
8. Severity & Priority: Classify the severity and priority of this defect with reasoning.
9. Coverage Gaps : Identify any missing test coverage based on these test cases.
10. Test Summary: Create a concise test summary for release readiness.
