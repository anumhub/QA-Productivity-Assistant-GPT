User Story: Password Reset Feature

As a registered user,
I want to reset my password using my email address,
So that I can regain access if I forget my password.

Acceptance Criteria:
- User can request password reset from login page
- Reset link is sent to registered email
- Reset link expires after 30 minutes
- New password must meet security rules
- User can login successfully after reset

Assumptions:
- User is already registered
- Email service is configured
- Feature is available on web application
